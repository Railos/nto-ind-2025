# Решение индивидуальной задачки на направление ИИ НТО 2025

Для установки предпочтительно использовать `venv`
Чтобы установить необходимые пакеты можно использовать `pip`
`pip install -r requirements.txt`

Далее просто запустить main.py
На выходе будет файл submission.csv который будет являться сабмитом.
